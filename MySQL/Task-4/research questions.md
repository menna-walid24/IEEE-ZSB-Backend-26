**1-UNION vs UNION ALL:**
Union and union all both combine results of 2 or more queries
**But the difference is:**
1-Duplicate Handling:
union return only distinct values
union all return duplicates
**2-performance:**
Union is slower than union all because the process of removing duplicates
***
**2. Subquery vs JOIN:**
1-join is generally faster than a subquery, especially for large tables ,in this
case join is better than subquery.
2-join is preferred if we need to compine multiple tables
